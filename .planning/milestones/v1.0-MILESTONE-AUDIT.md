### Gap Closure (Phase 7)

**Previous Gap:** install.sh was missing cp command for `.kimi/agents/*.md` files

**Fix Applied:** Phase 7 Plan 01 added the following to `install.sh` lines 469-473:
```bash
# Copy Kimi agent system prompts (MD files)
if [ -d "$SCRIPT_DIR/.kimi/agents" ] && ls "$SCRIPT_DIR/.kimi/agents/"*.md &> /dev/null 2>&1; then
    cp "$SCRIPT_DIR/.kimi/agents/"*.md "$TARGET_DIR/.kimi/agents/"
    echo -e "  ${GREEN}${NC} Copied Kimi agent prompts ($(ls -1 "$SCRIPT_DIR/.kimi/agents/"*.md 2>/dev/null | wc -l) files)"
fi
```

**Commit:** `a1144ec` - fix(07-01): add agent MD file copying to install.sh

**Result:** Installation now copies both YAML configs and MD system prompts. Post-install role invocation works correctly.
